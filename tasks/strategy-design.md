# 乩童賺錢策略閉環系統設計

## 目標
讓收入能透過「策略 + 疊加」指數成長，而非線性。豪宅(5億)才追得到。變成策略養成遊戲。

## 核心閉環
香火收入 → 投資「事業升級（法器/分壇/行銷）」→ 乘數疊加 → 收入指數成長 → 更大投資／買奢侈品炫耀
（奢侈品＝純炫耀終點；事業升級＝再投資引擎）

## 四個乘數來源（相乘，造成指數）
1. **rep 名聲** → 人數 × 大方度（既有）
2. **神明技能 (god skill)** → 每位神明一個被動 + 一個主動技
3. **事業升級 (upgrades)** → 永久乘數，用香火買，可疊加
4. **連段 Combo / 心流** → 一班內連續「對症好答」累積 combo，香火倍率即時飆升

收入 ≈ 基礎 × repMult × godPassive × Π(upgrades) × comboMult
→ 任一項提升都讓全局乘數放大 = 指數感。

## 神明技能（每位不同能力，鼓勵換神打法）
- 三太子（爆發型）：被動 combo 升得快；主動「陣頭」一次答的香火 ×3（限本班一次）。擅長：討債/感情→高額。
- 關聖（穩定型）：被動 滿意度不易掉、rep 增益 +50%；主動「義氣」本班開銷減半。適合衝名聲。
- 媽祖（保本型）：被動 元神消耗減半（可多答幾輪）；主動「庇佑」本班不會被鬧場/掉粉。適合穩穩刷。
- 濟公（賭徒型）：被動 香火隨機波動大（高方差，可暴擊 ×5）；主動「醉拳」下一題三選項都變高香火但元神狂掉。高風險高報酬。
- 王爺（財閥型）：被動 香火 +25% 但滿意度更難拉；主動「開壇」本班香火直接 ×1.5 結算。純斂財最強。

每位有 cooldown / 每班可用次數，玩家要「看信眾組成選神 + 抓時機放技能」。

## 事業升級（新商店分頁「廟務經營」，用香火買，永久乘數）
階梯式、可疊加、越買越貴但回本：
- 擴建香爐 Lv1–5：每級 香火基礎 +15%
- 行銷小編 Lv1–5：每級 rep 增益 +12%（名聲漲更快 → 滾雪球）
- 分壇加盟 1–5 間：每間 每班被動收入（離線/掛機感）+ 人數上限
- VIP 包廂：高價信眾出現率 ↑
- 法師團隊：元神上限 ↑（能答更多輪 = 更多香火）
- 神明加持（解鎖第二尊神同台）：兩位神被動疊加
這些是「再投資引擎」，讓滾雪球成形。

## Combo / 心流（一班內即時疊加）
- 連續對症好答 → combo+1，香火倍率 = 1 + combo×0.15（封頂可由升級拉高）
- 亂答/坑錢 → combo 歸零
- 鼓勵「好好答衝 combo」與「斂財爆收入」兩種流派拉扯（斂財給錢但斷 combo 且掉 rep）

## 難度與平衡
- 早期：乘數都低，淨利微薄（維持現有壓力感）。
- 中期：買幾個升級後開始正循環。
- 後期：多乘數相乘 → 指數起飛，買得起帝寶/棕櫚島。
- 公式集中 economy.js / 新 upgrades.js，好調。

## 存檔擴充（store schema）
- save.upgrades = { incenseLv, marketingLv, branches, vip, staff, secondGod }
- save.skills = { [godId]: { cooldownLeft } }（跨班 cooldown 可選）
- 既有欄位不動，migrate 補預設。

## UI
- 商店加分頁「廟務經營」（升級樹，顯示目前乘數與下一級效果/價格）。
- 值班 HUD 顯示 combo 連段數與當前香火倍率（即時跳動爽感）。
- 神明技能按鈕（值班中可放主動技）。
- 結算顯示「總乘數拆解」讓玩家看懂自己的 build。
